    Tutorial: dat110-project2-javafx-chatapp

    test This project is a simple JavaFX Chat Application (ChApp). It allows users to connect to a central message broker (server), subscribe to chat topics, and publish/receive messages on those topics. The application has a graphical user interface (GUI) for interaction and handles the underlying network communication and message formatting.
